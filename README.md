# CodeAlpha Basic Chatbot

A simple rule-based chatbot developed using Python as part of my CodeAlpha Python Programming Internship.

## Features

- Responds to greetings and basic questions
- Provides predefined responses
- Handles unknown messages
- Continues chatting until the user types "bye"

## Concepts Used

- Python functions
- If-elif conditions
- While loops
- Lists
- User input and output

## How to Run

1. Download the project.
2. Open the project folder in VS Code.
3. Run:

```bash
python chatbot.py
